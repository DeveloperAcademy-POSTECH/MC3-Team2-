# :iphone: APillog

### 주의력 결핍, 과잉행동 증세를 겪는 사람들을 위한 기록 앱입니다.


## :pushpin: Features

- 간편한 약 복용 기록
- 약 효능, 부작용 기록 및 트래킹
- 실수 노트를 통한 자기 인지 행동 치료 보조
- 진료 때 자료로 사용할 수 있는 기록 모아보기
